 Quai Antique - Site Vitrine

Projet réalisé dans le cadre de ma formation Studi.

Description
Création d’un site vitrine responsive pour le restaurant Quai Antique.

Le projet met en avant :
- la présentation du restaurant
- les menus
- les informations de contact
- une navigation fluide et responsive

# Technologies utilisées
- HTML5
- CSS3
- JavaScript

 Objectifs pédagogiques
- Structurer une page web moderne
- Créer un design responsive
- Utiliser JavaScript pour l’interactivité
- Organiser un projet front-end

 Lancer le projet
1. Télécharger le projet
2. Ouvrir `index.html` dans un navigateur
